# Listview App

(Properties: Name, Code, Description, Completion Date) 

#### Description
 - List/Table representation of components
 - *Name* and *Code* are view only
 - *Description* and *Completion Date* should be editable
 - Items in the list should be selectable
 - On select the item should expand in a way to display a little process graph (http://upload.wikimedia.org/wikipedia/de/thumb/3/33/NetzplanBsp.png/800px-NetzplanBsp.png)
